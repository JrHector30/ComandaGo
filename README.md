# ComandaGo - Instalación

## Requisitos
- Node.js 18+
- MySQL/MariaDB corriendo (Nota: El proyecto usa SQLite por defecto en dev.db, pero si usas MySQL asegúrate de tenerlo).

## Instalación rápida
1. npm install
2. npx prisma db push && node seed.js
3. npm start
4. Abrir http://localhost:3000
